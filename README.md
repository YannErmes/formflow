# FormFlow - Forms, Emails & Remarks Manager

FormFlow is a comprehensive management tool for creating, organizing, and sharing forms, email templates, and remark templates all in one place.

## Features

- **Form Builder** - Create dynamic form fields with tags, organize by categories
- **Email Templates** - Build and manage email templates with predefined content
- **Remark Templates** - Create remark templates with placeholders for quick text generation
- **Import/Export** - Backup and share your templates as JSON files
- **Local Storage** - All data is stored locally in your browser
- **Dark & Light Mode** - Beautiful black and white interface

## Getting Started

### Prerequisites
- Node.js 16+
- npm or bun

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Usage

1. **Forms**: Create reusable form fields and fill them out with tag-based organization
2. **Emails**: Create email templates with subjects, recipients, and body content
3. **Remarks**: Create templates with `{placeholder}` or `"placeholder"` syntax for dynamic content

## Export & Import

- Export templates from any section to share with colleagues
- Import templates from friends or colleagues - they'll merge with your existing data
- Use JSON format for easy backup and restoration
